# 查询目中的内容信息

语法：
```
ls [选择] [文件或目录]
```
选项：
- a 显示所有文件，包括隐藏文件
- l 显示详细信息
- d 查看目录信息
- h 显示文件大小（人性化）
- i 显示inode


# 搜索文件 
语法：
```
locate 文件名
```

# 命令搜索命令
语法：
```
whereis 命令名
或
which 命令名
```
-b 只查找可执行文件
-m 只查找帮助文件

# 文件搜索
```
find [搜索范围] [搜索条件]
```
搜索条件：
* 匹配任意内容
? 匹配任意一个字符
[] 匹配任意一个中括号内的字符


# 搜索字符串
语法：
```
grep [选项] 字符串 文件名
```
选项：
- i 忽略大小写
- v 排除指定字符串
