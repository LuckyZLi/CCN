总操作流程：
- 1、下载安装
- 2、搭建大型单页应用
- 3、安装并运行
- 4、测试

----------

# 下载安装
```
cnpm install vue -g
```
# 搭建大型单页应用
```
cnpm install --global vue-cli -g # 全局安装 vue-cli

vue init webpack my-project # 创建一个基于 webpack 模板的新项目

剩下的默认就行
```
# 安装并运行
```
cd my-project
cnpm install
cnpm run dev
```
# 测试
访问 http://localhost:8080/

**ctrl+c ：退出**
